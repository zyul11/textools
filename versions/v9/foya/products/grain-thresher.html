Grain Thresher

The FOYA grain thresher is a versatile machine designed to thresh multiple types of grain crops. It separates grain from stalks, husks and chaff efficiently, delivering clean grain ready for storage or sale.

The machine features adjustable drum speed and concave clearance to accommodate different crops. It can be powered by an electric motor, diesel engine or tractor PTO.

📋 Key Specifications

  • Capacity: 500-3000 kg/hour (crop dependent)
  • Power: 3-11 kW (electric) or 5-20 hp (diesel)
  • Threshing Drum Speed: 500-1200 rpm (adjustable)
  • Cleaning: Fan and sieve system
  • Grain Loss: < 1.5%
  • Weight: 150-400 kg

✨ Product Features

  • Multi-crop capability — wheat, rice, corn, beans
  • Adjustable drum speed for different crops
  • Efficient cleaning system removes chaff
  • Low grain loss rate
  • Portable design for farm use
  • Simple operation and easy maintenance
