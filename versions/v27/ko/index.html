단어 카운터

텍스트의 단어 수, 문자 수, 문장 수, 문단 수, 읽기 시간을 계산합니다.
🧹

중복 줄 제거

텍스트에서 중복된 줄을 제거합니다. 목록, CSV 데이터, 반복 항목을 정리하세요.
🔤

줄 정렬

텍스트 줄을 알파벳순으로 정렬합니다 (A-Z 또는 Z-A). 목록과 데이터 정리에 유용합니다.
🔍

찾기 및 바꾸기

대소문자 구분 및 단어 전체 일치 옵션으로 텍스트를 찾고 바꿉니다.
🔄

대소문자 변환

텍스트를 대문자, 소문자, 첫 글자 대문자, 문장 형식으로 변환합니다.
📏

텍스트 뒤집기

텍스트, 단어 또는 줄을 뒤집습니다. 거울 문자와 거꾸로 텍스트를 생성합니다.
