🧰 텍스트 도구

무료 온라인 텍스트 처리 — 모든 도구가 브라우저에서 실행되며, 데이터가 외부로 전송되지 않습니다

🔒 100% 프라이버시 보호 — 모든 텍스트 처리는 브라우저에서 로컬로 실행됩니다. 서버에 업로드되는 데이터는 없습니다.

📊

단어 카운터

텍스트의 단어 수, 문자 수, 문장 수, 문단 수, 읽기 시간을 계산합니다.
🔢

문자 카운터

공백 포함/미포함 문자 수 계산. Twitter, SMS, 메타 설명에 최적.
📝

Markdown 미리보기

Markdown을 실시간 작성/미리보기. 표, 코드 블록, 서식 지원.
🔧 텍스트 처리
🗑️

중복 줄 제거

텍스트에서 중복된 줄을 제거합니다. 원래 순서 유지.
🔤

줄 정렬

텍스트 줄을 알파벳순으로 정렬 (A-Z 또는 Z-A).
🔄

대소문자 변환

대문자, 소문자, 첫 글자 대문자, 카멜케이스 등으로 변환.
🔍

찾기 및 바꾸기

텍스트 검색 및 바꾸기. 정규식 모드 지원.
↩️

텍스트 뒤집기

텍스트, 단어, 줄을 뒤집습니다. 회문과 퍼즐에 유용.
✂️

텍스트 분할

구분 기호, 줄 바꿈, 정규식 패턴으로 텍스트 분할.
📋

텍스트 차이 비교

두 텍스트를 나란히 비교. 추가/삭제/변경 강조.
🧹

공백 문자 제거

여분의 공백, 탭, 줄 바꿈을 제거하여 텍스트 정리.
🔢

줄 번호 매기기

텍스트에 줄 번호를 추가. 접두사/접미사/시작 번호 사용자 지정.
📄

템플릿 도구

사용자 지정 값으로 텍스트 템플릿 채우기. 재사용 가능한 패턴 생성.
🔣 인코딩 및 변환
🔐

Base64 인코더/디코더

텍스트를 Base64로 인코딩/디코딩. 파일 입력 지원.
🔗

URL 인코더/디코더

URL 매개변수 인코딩/디코딩. 특수 문자 처리.
📌

URL 슬러그 생성기

텍스트에서 URL 친화적 슬러그 생성. 블로그 게시물에 최적.
⏰

타임스탬프 변환기

Unix 타임스탬프와 날짜 상호 변환. 여러 시간대 지원.
📋

JSON 포맷터

JSON 데이터 포맷팅, 검증 및 예쁘게 출력. 트리 뷰 접기 가능.
🛠️ 유틸리티
🔐

비밀번호 생성기

강력한 랜덤 비밀번호 생성. 길이, 기호, 숫자 사용자 지정.
🔬

정규식 테스터

실시간 정규식 테스트. 일치, 그룹, 대체 결과 표시.
🎲

랜덤 선택

목록에서 항목을 무작위로 선택. 추첨, 제비뽑기, 팀 구성에 유용.
