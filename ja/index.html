🧰 テキストツール

無料のオンラインテキスト処理 — すべてのツールはブラウザ上で動作し、データは端末から外部に出ません

🔒 プライバシー完全保護 — すべてのテキスト処理はお使いのブラウザ上でローカルに実行されます。サーバーにデータが送信されることは一切ありません。
📝 テキスト分析
📊

単語カウンター

文字数・単語数・文章数・段落数・読了時間をリアルタイムで集計します。
🔢

文字数カウンター

スペースを含む／含まない文字数をカウント。Twitter・SMS・メタディスクリプションに最適。
📝

Markdown プレビュー

リアルタイムでMarkdownを記述・プレビュー。テーブル・コードブロック・書式に対応。
🔧 テキスト処理
🗑️

重複行削除

テキストから重複行を削除。元の順序を保持したまま一括除去。
🔤

行ソート

テキスト行を昇順（A-Z）または降順（Z-A）に並べ替え。
🔄

大文字小文字変換

大文字・小文字・タイトルケース・キャメルケースなどに変換。
🔍

検索・置換

テキストを検索して置換。正規表現モード対応。
↩️

テキスト反転

文字列・単語・行を反転。パリンドロームやパズルに。
✂️

テキスト分割

区切り文字・改行・正規表現でテキストを分割。
📋

差分チェッカー

2つのテキストを並べて比較。追加・削除・変更をハイライト。
🧹

空白文字削除

余分なスペース・タブ・改行を削除。テキストをクリーンアップ。
🔢

行番号付与

テキストに行番号を追加。接頭辞・接尾辞・開始番号をカスタマイズ。
📄

テンプレートツール

カスタム値でテキストテンプレートを埋め込み。再利用可能なパターンを作成。
🔣 エンコード・変換
🔐

Base64 エンコーダ/デコーダ

テキストをBase64形式にエンコード／デコード。ファイル入力にも対応。
🔗

URL エンコーダ/デコーダ

URLパラメータをエンコード／デコード。特殊文字を処理。
📌

URL スラグ生成

テキストからURLに適したスラグを生成。ブログ記事やページに最適。
⏰

タイムスタンプ変換

Unixタイムスタンプと日付を相互変換。複数のタイムゾーン対応。
📋

JSON フォーマッター

JSONデータをフォーマット・検証・美化。ツリービューで折りたたみ可能。
🛠️ ユーティリティ
🔐

パスワード生成

高強度ランダムパスワードを生成。長さ・記号・数字をカスタマイズ。
🔬

正規表現テスター

リアルタイムで正規表現をテスト。マッチ・グループ・置換結果を表示。
🎲

ランダム選択

リストからランダムに項目を選択。抽選・くじ引き・チーム分けに。
